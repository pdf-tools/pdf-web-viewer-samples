### Key Design Decisions

| Decision | Reason |
|----------|--------|
| `port: 0` | OS assigns available port (avoids conflicts) |
| `hostname: 127.0.0.1` | Loopback only - server not accessible from network (security) |
| Reuse existing server | Prevents duplicate servers on hot reload |
| Mounted check | Prevents state updates on unmounted component |

### Return Values

```javascript
const { serverUrl, isReady, error } = usePdfViewerServer();
```

| Value | Type | Description |
|-------|------|-------------|
| `serverUrl` | `string \| null` | e.g., `http://127.0.0.1:54321` |
| `isReady` | `boolean` | True when server is serving |
| `error` | `Error \| null` | Set if startup failed |
