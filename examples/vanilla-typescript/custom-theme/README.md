# Custom themes

This is an example that demonstrates the basic usage of the PDF Web Viewer, with the custom themes.
In the example, we demonstrate the usage of the `ViewerConfig.customThemes` property and the following methods available on the `PdfToolsViewer` interface:

- `getTheme`: Gets the currently applied theme.
- `setTheme`: Sets the theme.
- `getAvailableThemes`: Gets the list of all available themes.
- `addTheme`: Adds a new theme.
