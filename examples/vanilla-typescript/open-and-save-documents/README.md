# Open and save documents

This is an example that demonstrates the basic usage of the PDF Web Viewer, where the open document and save document buttons have a custom behavior.
In this example, these buttons switch behavior:

- When the user clicks on a save button, it opens a new PDF document programmatically, by using the `Viewer.document.open` method.
- When the user clicks on the open document button, it downloads the PDF document programmatically, by using the `Viewer.document.download` method.
